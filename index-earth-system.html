Welcome to Galaxy for Earth System and environment


Galaxy for Earth System and environment was implemented within the project Fair-Ease. It‚Äôs a virtual platform to process, analyse and visualize Earth System, Environment and Biodiversity data. It is based on the Galaxy framework, which guarantees simple access, easy extension, flexible adaption to personal and security needs, and sophisticated analyses independent of command-line knowledge.

Tools

Multiple tools on Earth System, Environment and Biodiversity data analysis will be available in this custom Galaxy instance:

  • General tools
    • QGIS
    • HDFviewer
  • Coastal Water Dynamics tools
    • Ocean Data viewer ODV
    • DIVAnd
    • Source
  • Bio-GeoChemical tools
    • Scoop Argo
  • Marine omics tools
  • Earth critical zone tools on land degradation
  • Volcano tools
